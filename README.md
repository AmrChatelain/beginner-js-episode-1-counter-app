# 🔢 Counter App (Stops at 0)

## 📌 About The Project

Welcome to **Episode 1** of my Beginner JavaScript Series 🚀

In this project, we build a simple **Counter App** where:

- The number increases when clicking the **Up** button
- The number decreases when clicking the **Down** button
- The counter **never goes below 0**

This is a beginner-friendly project focused on understanding JavaScript fundamentals and DOM manipulation.

### 💡 How It Works

- We store the current number inside a variable called `current`.
- When the **Up** button is clicked, we increase the number by 1.
- When the **Down** button is clicked, we decrease the number by 1.
- If the number becomes less than 0, we reset it back to 0.
- Then we update the display using `textContent`.

Simple logic. Clean structure. Beginner-friendly.
